## Method 1: Vercel CLI (Fastest)

1. **Install Vercel CLI:**
   ```bash
   npm install -g vercel
   ```

2. **Login and Deploy:**
   ```bash
   vercel login
   vercel --prod
   ```

3. **Done!** You'll get a live URL instantly.

---

## Method 2: Vercel Website (No CLI needed)

1. **Go to:** https://vercel.com
2. **Sign up** with GitHub
3. **Import** your GitHub repo: `School_forum`
4. **Click Deploy**
5. **Done!** Auto-deployed with custom URL

---

## Method 3: Netlify (Drag & Drop)

**For Frontend Only:**
1. **Build frontend:**
   ```bash
   cd frontend
   npm install
   npm run build
   ```

2. **Go to:** https://app.netlify.com/drop
3. **Drag the `build` folder**
4. **Get instant URL!**
